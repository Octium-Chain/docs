    Trezor User Guide

    This guide will show you how to use your Trezor Model T hardware wallet with OCT Chain and OCT DEX. Please follow best security practices when using any hardware wallet to store cryptocurrency.

    Please note that the Trezor Model One is not supported yet.

    Requirements

    In order to use a Trezor with OCT Chain and OCT DEX, you will need: * A supported web browser: Chrome or Firefox * Access to the OCT Chain web wallet (https://scan.octium.io/unlock) using your web browser * An initialized Trezor Model T device with the most recent firmware updates installed * The Trezor Bridge application installed on your computer

    App Installation Instructions

    1. Install Trezor Bridge

    Please make sure that you have bridge installed on your computer.

    1. Install the firmware

    Please make sure your Trezor has the latest firmware installed.

    1. Generate and backup your address

    Please make sure that you have seed phrase backed up.
